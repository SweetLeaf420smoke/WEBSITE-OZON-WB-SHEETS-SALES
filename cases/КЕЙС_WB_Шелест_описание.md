# Кейс: WB-таблица (два заказа)

С этого проекта по сути началась тема «автоматизация WB в Google Sheets» и потом лендинг.

**Факт:** два заказа.

**Таблица:** https://docs.google.com/spreadsheets/d/1NsWDoq6H9eYKz6Z8Z9yYLGqAZ_rhzfBZWux5PY3DdnU/edit

**Что в таблице:**
- Загрузка из API WB: продажи (v3), заказы (v1), остатки (v1), цены (upload).
- Листы: Report (сводка по складам), RAW_Sales, RAW_Orders, RAW_Stock.
- WarehouseRegions — склады и регионы; «Ближайший склад» по региону.
- Supply_Plan_By_ZONES — план поставок по зонам.
- Sales_By_SKU_Zone, Stock_By_SKU_Zone, Sales_by_Warehouse.
- Управление ценами по расписанию (меню, триггеры, пачки загрузки).
- WB_Prices, Price_List, Price_Schedule.
- Расчёт экономики: комиссия, СПП, логистика, НДС, прибыль, ROI; лист ПРОГНОЗ.

**Итог:** полноценная система аналитики, планирования поставок и управления ценами для селлера WB. Эта же схема (зоны, план по зонам, перераспределение) потом использовалась как эталон для таблицы перераспределения по ФО и для описания на сайте.
