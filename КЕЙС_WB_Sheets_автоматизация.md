# Подключение и автоматизация данных Wildberries в Google Sheets

Система аналитики продаж, планирования поставок и управления ценами

---

## Задача

Селлеру на Wildberries было необходимо решить несколько практических задач, которые возникают при масштабировании продаж на маркетплейсе:

- получать данные о продажах автоматически
- понимать, с каких складов и в какие регионы уходит товар
- планировать поставки по складам на основе реальных продаж
- контролировать остатки
- автоматически управлять ценами по расписанию
- рассчитывать экономику продаж

При ручной работе с кабинетами Wildberries и выгрузками это занимает огромное количество времени: данные разбросаны по разным отчетам, часть информации доступна только через API, а прогнозирование поставок и экономики приходится считать вручную.

Для решения этой задачи была разработана автоматизированная система аналитики и управления данными Wildberries на базе Google Sheets с подключением к API.

---

## Что было реализовано

### Полная интеграция Google Sheets с API Wildberries

Система подключается напрямую к API Wildberries и автоматически загружает данные в таблицу. Это избавляет от ручных выгрузок отчетов и позволяет работать с актуальной информацией.

В таблицу автоматически подгружаются:

- данные о продажах
- данные о заказах
- остатки по складам
- данные по ценам

Для удобства работы в интерфейс таблицы была добавлена панель управления с кнопками:

- **Wildberries – загрузка продаж** — автоматически заполняет лист RAW_SALES за выбранный период (7, 14, 21 или 30 дней)
- **Wildberries – загрузка остатков** — загружает полный список остатков по всем складам WB на лист RAW_STOCK

### Система обработки и структурирования данных

Сырые данные из API сохраняются в отдельных листах, после чего обрабатываются формулами и скриптами.

Основные листы системы:

- **RAW_SALES** — исходные данные продаж
- **RAW_STOCK** — остатки по складам
- **WB_ORDERS** — данные заказов
- **SALES_BY_SKU_ZONE** — продажи по артикулам и регионам
- **SUPPLY_PLAN_BY_ZONES** — расчет плана поставок

### Аналитика продаж по складам и регионам

Для каждого артикула определяется: с какого склада отправлен, в какой регион заказ, сколько продано. Формируется таблица **SALES_BY_SKU_ZONE** (артикул, склад отправки, регион заказа, количество продаж).

Регионы объединяются в зоны: Центральная, Южная, Кавказ, Сибирь, Казахстан, Центральная Азия. Новые регионы WB автоматически появляются в таблице и могут быть вручную отнесены к зоне.

### Автоматический расчет плана поставок

На листе **SUPPLY_PLAN_BY_ZONES**: выбирается период (например 21 день), система считает продажи по регионам, агрегирует по зонам, рассчитывает прогноз потребления и формирует план поставок (например Краснодар — 15 ед., Котовск — 13 ед.).

### Система управления ценами

Через API Wildberries: автоматическое изменение цен по расписанию, подъем/снижение на заданный процент, откат через заданное время. Пример: +30% в 00:00, возврат в 03:00. Скрипты пакетной загрузки (до 200 товаров за запрос, разбивка на части).

### Автоматические триггеры

Триггеры Google Apps Script для автоматического запуска: обновление продаж, остатков, изменение цен, расчеты (ночное изменение цен, откат, обновление аналитики).

### Расчет экономики продаж

Автоматический расчет по каждой позиции: чистая прибыль, ROI, налоги, комиссия WB, СПП, логистика. На основе уведомления о выкупе и детализированного отчета. Для каждой позиции: выручка, комиссия, СПП, логистика, закупка, упаковка, налоги, НДС входящий/исходящий/к уплате, прибыль, ROI.

Лист **ПРОГНОЗ** — моделирование экономики при изменении цены, логистики, закупки, комиссии (по какой цене продавать, какая прибыль, какой ROI).

### Автоматизация загрузки данных

Переход с ручной загрузки из файлов на загрузку напрямую через API WB: актуальные данные, без ручных выгрузок.

---

## Результат

Полноценная система управления данными WB для селлера:

- автоматическое получение данных из API
- аналитика продаж по складам и регионам
- прогноз поставок
- управление ценами по расписанию
- расчет экономики продаж и моделирование прибыли

Всё работает в Google Sheets, без сложного ПО. Система адаптируется под любой аккаунт WB и масштабируется с ростом бизнеса.
