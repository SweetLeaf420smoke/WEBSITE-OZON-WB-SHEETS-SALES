Таблица для Wildberries: перераспределение по складам WB и план поставок

Google-таблица для селлера WB: выгрузка заказов, остатков и продаж из API Wildberries — куда и сколько везти по складам, догрузка по ФО и артикулам. Таблица для маркетплейса под ваши задачи.

Получить таблицу

Для кого эта таблица для WB

  • Селлеры Wildberries с несколькими складами (Коледино, Питер, Екатеринбург, регионы) — кому нужна таблица для WB с выгрузкой данных.
  • Кому важно остатки по складам WB и управление догрузкой: не просто «есть ли товар», а где его не хватает относительно заказов.
  • Кто ищет перераспределение по складам Wildberries и хочет снизить долю нелокальных заказов — таблица показывает, куда везти по ФО и артикулам.

В чём проблема сейчас

  • В личном кабинете WB нет связки: заказы по складам, остатки по складам и сколько не хватает по ФО. Селлеры ищут таблицу для WB с выгрузкой — считают вручную.
  • Непонятно куда везти товар на Wildberries: на какой склад, по каким артикулам, план поставок по складам отсутствует.
  • Нелокальные заказы — дольше и дороже. Нужна таблица перераспределения по складам WB: заказы и остатки по ФО, рекомендации по догрузке.

Что даёт таблица для WB

Подключение к API Wildberries (токен Statistics API) — выгрузка из WB в таблицу: заказы, остатки, продажи. Обновление по кнопке, без ручной выгрузки.

  • Перераспределение по складам WB (свод по ФО) — по каждому округу: склады, заказы локальные/нелокальные, остатки по складам, сколько перераспределить, рекомендации по догрузке.
  • Перераспределение по артикулам — по ФО и артикулу: заказы, остаток, сколько штук догрузить. Только позиции с дефицитом.

Итог: план поставок по складам в цифрах — куда и сколько везти для решений по догрузкам.

Выгоды

  • Экономия времени — выгрузка из WB в таблицу по кнопке, не нужно вручную сводить заказы по складам и остатки по складам WB.
  • Прозрачность по ФО — видно, в каких округах не хватает товара и где много нелокальных заказов Wildberries.
  • План поставок по складам — готовые цифры «сколько перераспределить» по ФО и артикулам для догрузки.
  • Таблица для селлера — Google Таблица, кнопка «Обновить всё», без программ. Подключение API WB, только браузер.

Как пользоваться

  1. Получить таблицу и один раз настроить: подставить свой токен WB Statistics из ЛК WB.
  2. В меню таблицы нажать «Обновить всё» (рекомендуется раз в день или перед решениями по догрузкам).
  3. Открыть листы «Перераспределение» и «Перераспределение по артикулам» — смотреть дефицит и сколько перераспределить.
  4. Использовать данные для решений: какие склады догружать, в каком объёме и по каким артикулам.

Чем это не является

  • Это не система учёта или замена 1С/склада.
  • Это не автоматическая отгрузка — таблица только считает и показывает рекомендации; решение принимает продавец.
  • Это не прогноз спроса — работа с текущими заказами и остатками «здесь и сейчас».

Что получите

  • Доступ к копии таблицы (шаблон под свой токен) + краткая инструкция по подключению и обновлению.
  • Опционально: короткое обучение/созвон — как читать отчёты и использовать цифры при планировании догрузок.
