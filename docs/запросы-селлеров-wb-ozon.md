# Пул запросов селлеров WB и Ozon

## 1. Введение

Документ нужен для подбора формулировок под контент сайта и SEO: какие фразы и запросы используют селлеры Wildberries и Ozon, когда ищут таблицы, выгрузку из API, перераспределение по складам, план поставок, расчёт экономики по артикулам. Понимание этих запросов помогает отвечать на реальные формулировки клиентов и попадать в поиск.

Собрано по открытым источникам: форумы продавцов маркетплейсов, статьи и гайды, обзоры сервисов. Формулировки из чатов Telegram и живых обсуждений можно дополнять вручную по мере просмотра.
